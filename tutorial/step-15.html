Step 15 - Deploy! Use the tool woc-make

Our application is finished!

Itâ€™s time to deploy it.

From command line, go to the project directory and run:

node _woctools/woc-make todos

This command makes a new directory todos-0.1 for the bundle todos in version 0.1. In this directory, there is a file todos.json that contains all the JavaScript code and all the HTML template code minified and merged. The CSS code is minified and merged in the file todos.css. The file ajax.gif comes from the theme SimpleTheme. The directory images comes from the external library jQueryUI.

Edit the file index.html. Replace the directive data-woc-load="todos(w)" with data-woc-load="todos(0.1,css)":

<div data-woc-load="todos(0.1,css)" data-woc-start="Todos.Start"></div>

Without the optional first parameter w (working mode), the deployed version is loaded. The 0.1 is the version. The last parameter css tells to Woc.ts to load the CSS during it loads the JSON file.

Reload the application in the browser: the deployed version of the application is loaded.

We can use a reduced version of the Woc.ts core, without the loader of working mode. Replace woc-w.min.js with woc.min.js:

<script src="woc.min.js"></script>

Reload the application in the browser.

The tutorial is finished.
